# EzStore Assignment (Backend)

## How to run

```
git clone this project
```

### Run with Docker

Open your Docker Desktop

```
./vendor/bin/sail up
```

Open another tab in the terminal/cmd

```
./vendor/bin/sail npm install && npm run dev
```

Then access http://127.0.0.1/ with browser
